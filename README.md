# csv2sqlite

This is a simple tool to convert a csv file into a SQlite Database.

## Installation

Step 1: If you have git CLI installed, run

git clone https://github.com/akbarali97/csv2sqlite.git

Or 

download, unzip and open load.py from https://github.com/akbarali97/csv2sqlite.git

Step 2: copy the csv file to the same directory of load.py

Step 3: run 
        python load.py (windows) 
                or 
        python3 load.py (Linux, Mac)

Step 4: Enter the name of the csv file into the Terminal and hit enter.

Step 5: A SQlite database will be generated!
